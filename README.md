# NFT-Platformer
A unity platformer for the playbitrace.com game

upcomming merge into the wix playbitrace.com website 
Must purchase a bitrace NFT to play the game and earn bte

This is a Unity game, scale the terrain islands with double jumps that reset when you land on something (you may use your NFT car as a surface to reset)

note that the assets folder and library folders are large, so you must unzip them.

Enjoy playing/testing!
